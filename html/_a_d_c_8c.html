Detailed Description

Provides functions for utilizing the ADC converter.

This source file contains a library of functions to utilize the ADC in both polling and interrupt mode

Author:
Adam Panzica
Joel Sotherland
Date:
22-Jan-2009
Version:
1.0 Initial version with ADC_Init_P() and ADC_Get_Value()
1.1 Updated ADC code, fixed prescaler issue, added ADC_Sensor struct and its constructor ADC_Sensor_Construct() for conversions.
1.2 Added ADC_Sensor_Calc_Value() and ADC_Calc_Volts()
