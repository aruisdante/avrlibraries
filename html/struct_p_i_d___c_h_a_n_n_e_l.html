Detailed Description

this strcut defines a PID_CHANNEL object

The PID_CHANNEL serves as an interface to the PID controller. All interactions with the PID controller happen through a PID_CHANNEL struct. Specifically, the Kp, Ki, Kd and spool constants are set once, and then the contents of the variables pointed to by the 'Communication Variables', *target and *current, can be used to update the PID controller, while *correctionValue and *LMDValue can be used to monitor status. The 'Internal Variables' are utilized by 'private' functions in the PID controller, and thus should not be accessed by the user directly.
